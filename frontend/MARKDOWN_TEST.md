# Markdown 渲染測試說明

## 測試方法

1. 打開瀏覽器訪問 http://localhost:3002/
2. 在聊天輸入框中請求 AI 回覆包含以下 Markdown 元素的訊息

## 測試案例

### 1. 基本格式測試
請 AI 回覆：
```
請給我一個包含**加粗**、*斜體*和~~刪除線~~的範例
```

### 2. 列表測試
請 AI 回覆：
```
請列出三個要點：
1. 第一點
2. 第二點
3. 第三點
```

### 3. 表格測試
請 AI 回覆：
```
請給我一個簡單的表格範例
```

期望看到類似：
| 欄位 1 | 欄位 2 | 欄位 3 |
|--------|--------|--------|
| 資料 1 | 資料 2 | 資料 3 |

### 4. 程式碼區塊測試
請 AI 回覆：
```
請給我一個 JavaScript 函式範例
```

期望看到程式碼區塊有適當的背景色和語法顯示。

### 5. 連結測試
請 AI 回覆包含連結的訊息

### 6. XSS 安全性測試
嘗試發送：
```
<script>alert('XSS')</script>
```

應該被正確清理，不會執行腳本。

## 驗證項目

- ✓ AI 訊息的 Markdown 正確渲染
- ✓ 使用者訊息保持純文本顯示
- ✓ 表格、列表、加粗、標題等格式正確
- ✓ 程式碼區塊有適當的背景色
- ✓ 惡意 HTML/JavaScript 被過濾
- ✓ 整體樣式美觀且易讀
